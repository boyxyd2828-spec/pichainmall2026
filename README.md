# PICHAINMALL FINAL VERCEL

File final siap upload ke GitHub dan deploy ke Vercel.

## Cara pakai paling mudah

1. Download ZIP.
2. Extract ZIP.
3. Upload semua isi folder ke GitHub.
4. Di Vercel pilih Import GitHub Repository.
5. Gunakan setting:
   - Framework: Vite
   - Install Command: npm install
   - Build Command: npm run build
   - Output Directory: dist

## Jalankan lokal

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
